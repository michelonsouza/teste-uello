# uello

## Project setup

```
yarn install
```

## .env file config

##### Rename `.env.example` to `.env`

##### Edit `.env` file

```js
VUE_APP_GOOGLE_MAPS_KEY=<YOUR_GOOGLE_API_KEY>
VUE_APP_HOST=<YOUR_HOST> // http://localhost:8080 for default local running app
```
